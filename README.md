# Academy SVG Icon Sprite

The svg should always be placed inside of a parent element with the class `.icon`:
```
<div class="btn">
	<svg class="icon">
		<use xlink:href="#icon-name" />
		//href should reference the ID of whatever icon you need to display
	</svg>
</div>
```

Icons:
-add-circle
-caret-left
-caret-right
-invodo-360
-subtract-circle
-times-circle
